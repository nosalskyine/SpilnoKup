# СпільноКуп — iOS Native App

## Про проект

СпільноКуп — це додаток для спільних покупок. Існує у двох версіях:
- **Веб-версія (PWA)** — у кореневій папці (`src/`, React + Vite)
- **iOS-версія (нативна)** — у папці `ios/` (SwiftUI, цей проект)

iOS-версія повинна повторювати функціонал веб-версії, але як нативний додаток для iPhone.

## Що вже є у веб-версії (src/App.jsx)

Веб-версія містить повноцінний функціонал:
- **Головна/Маркет** — каталог спільних покупок з фільтрами (категорія, місто, ціна, знижка, рейтинг), пошуком, сортуванням
- **Картки товарів** — фото, ціна гуртова/роздрібна, прогрес-бар зібраних учасників, HOT-бейджі
- **Деталі угоди** — повна інформація, карта, учасники, кнопка приєднатись
- **Створення угоди** — форма з назвою, описом, цінами, категорією, містом, фото, завантаження фото
- **Месенджер/Чат** — список чатів з продавцями
- **QR-код** — генерація та сканування QR для підтвердження покупки
- **Профіль/Гаманець** — баланс, поповнення, бонуси, реферальна програма, налаштування
- **Бізнес-панель продавця** — дашборд з аналітикою, управління товарами
- **Мої покупки** — історія замовлень
- **3 теми оформлення** — Графіт, Океан, Слива (темні теми)
- **Реєстрація/вхід** — email, телефон, SMS-код
- **Гостьовий режим** — перегляд без реєстрації
- **HOT-слайдер** — карусель гарячих пропозицій
- **Карта** — симуляція розташування точок видачі

## Що є в iOS-версії зараз

Поки що тільки базова оболонка:
- SwiftUI додаток з TabView (5 вкладок)
- Вкладки: Головна, Каталог, Створити, Сповіщення, Профіль
- Кожна вкладка — заглушка з NavigationStack та текстом
- Акцентний колір — оранжевий
- Deployment target: iOS 16.0
- Сумісність: Xcode 14.3.1+

## Обмеження середовища

- Тестування відбувається на MacBook з **Xcode 14.3.1** (macOS 13)
- НЕ використовувати `#Preview` — тільки `PreviewProvider`
- НЕ використовувати API з iOS 17+ (`Tab()`, `Observable`, `@Bindable` тощо)
- Використовувати `.tabItem {}` замість `Tab()` для TabView
- НЕ використовувати `ASSETCATALOG_COMPILER_GENERATE_SWIFT_ASSET_SYMBOL_EXTENSIONS`, `ENABLE_USER_SCRIPT_SANDBOXING`, `LOCALIZATION_PREFERS_STRING_CATALOGS` в build settings

## Як запустити

1. Відкрити `ios/SpilnoKup.xcodeproj` в Xcode
2. Обрати симулятор iPhone (напр. iPhone 14)
3. Cmd+R для запуску

## Наступні кроки

Поступово переносити функціонал з веб-версії в нативний iOS додаток, починаючи з:
1. Моделі даних (Deal, User)
2. Картки товарів на головній сторінці
3. Детальна сторінка угоди
4. Форма створення угоди
5. Профіль та авторизація
